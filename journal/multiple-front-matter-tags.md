---
title: "Two ways we can set multiple 'tags' in our front matter"
date: 2022-04-12
---

1. We can set multiple `tags` in a single line
    ```yaml
    title: Eleventy tags
    tags: ["eleventy", "web-development"]
    ```
2. Or in multiple lines
    ```yaml
    title: Eleventy tags
    tags:
        - eleventy
        - web-development
    ```
